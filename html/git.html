Git

  • git config - Настраивает различные параметры конфигурации.
  • git init - Инициализирует новый локальный репозиторий Git.
  • git status - Показывает статус изменений в рабочем каталоге.
  • git add - Переносит ожидающие изменения из рабочего каталога в раздел проиндексированных файлов.
  • git commit - Сохраняет изменения в репозитории.
  • git clone - Создает копию существующего репозитория Git.
  • git pull - Обновляет локальную копию репозитория с удаленного репозитория.
  • git push - Отправляет локальные изменения в удаленный репозиторий.
  • git log - Отслеживает изменения в репозитории.
